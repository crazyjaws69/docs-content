**Description**

Converts a value to the `link:../../data-types/char[char]` data type.

**Syntax**

`char(x)`
`(char)x` (C-style type conversion)

**Parameters**

`x`: a value. Allowed data types: any type.

**Returns**

Data type: [`char`](../../data-types/char).
