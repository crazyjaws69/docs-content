**Description**

Converts a value to the `link:../../data-types/long[long]` data type.

**Syntax**

`long(x)`
`(long)x` (C-style type conversion)

**Parameters**

`x`: a value. Allowed data types: any type.

**Returns**

Data type: [`long`](../../data-types/long).
