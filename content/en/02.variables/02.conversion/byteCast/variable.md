**Description**

Converts a value to the `link:../../data-types/byte[byte]` data type.

**Syntax**

`byte(x)`
`(byte)x` (C-style type conversion)

**Parameters**

`x`: a value. Allowed data types: any type.

**Returns**

Data type: [`byte`](../../data-types/byte).
