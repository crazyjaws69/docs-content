**Description**

Tests whether or not a String starts with the characters of another
String.

**Syntax**

`myString.startsWith(myString2)`

**Parameters**

`myString, myString2`: a variable of type `String`.

**Returns**

`true`: if myString starts with the characters of myString2.
`false`: otherwise
