**Description**

Get a version of the String with any leading and trailing whitespace
removed. The `trim()` function modifies the String in place rather than
returning a new one.

**Syntax**

`myString.trim()`

**Parameters**

`myString`: a variable of type String.

**Returns**

Nothing
