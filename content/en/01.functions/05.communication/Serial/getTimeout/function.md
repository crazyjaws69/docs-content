**Description**

`Serial.getTimeout()` returns the timeout value set by
`Serial.setTimeout()`.

`Serial.getTimeout()` inherits from the [Stream](../../stream) utility
class.

**Syntax**

`_Serial_.getTimeout()`

**Parameters**

None

**Returns**

The timeout value set by `Serial.setTimeout()`. Data type:
`unsigned long`.
